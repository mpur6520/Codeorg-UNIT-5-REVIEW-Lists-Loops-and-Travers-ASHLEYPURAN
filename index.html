Lesson 10

In this lesson, we took a look at apps that use loops to traverse in order to gain an understanding of different ways traversals are used.

Explain how you would filter the dog dataset using traversal to have a filtered list of dogs who live long lives.

First, you would set a list up to have the information of the length of the dogs' lives as well as their names. You can get this information from the dog dataset by setting up a list to take the column of "maximum lifespan" and "names". You would also set up a list for the filtered dog names. Then, you would set up a regular for loop for a transversal where the index is equal to 0 and less than the length of the list (list.length), and that 1 is added to the index each time the loop runs. Inside this loop, you can set a prerequisite in order to filter out the longer lives successfully. For example, you can have a conditional like "if dogYears >= 14". Inside that conditional, you can use "appendItem" from the dogNames list to the filtered list, meaning that the name of the dog meeting the lifespan requirement is added to the other list containing the dog name by using "i" as an index. This can keep going through the list and filtering through it.
